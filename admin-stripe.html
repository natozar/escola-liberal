Deploy das Edge Functions Passo 4

Execute no terminal, na raiz do projeto:

npm install -g supabase
supabase login supabase link --project-ref hwjplecfqsckfiwxiedo
supabase secrets set STRIPE_SECRET_KEY=sk_test_SUA_CHAVE supabase secrets set STRIPE_WEBHOOK_SECRET=whsec_SUA_CHAVE
supabase functions deploy create-checkout supabase functions deploy stripe-webhook
Os arquivos create-checkout.ts e stripe-webhook.ts já estão no seu repositório. O Supabase CLI os detecta automaticamente na pasta raiz ou em supabase/functions/.
  • 1️⃣
    Acessar Supabase Edge Functions Supabase Dashboard → Edge Functions → New function
  • 2️⃣
    Criar função "create-checkout" Copiar o conteúdo de create-checkout.ts e colar no editor
  • 3️⃣
    Criar função "stripe-webhook" Copiar o conteúdo de stripe-webhook.ts e colar no editor
  • 4️⃣
    Configurar secrets via Dashboard Settings → Vault → Add Secret: STRIPE_SECRET_KEY e STRIPE_WEBHOOK_SECRET
