# Copyright & IP Specialist

## Role
Proteger propriedade intelectual e garantir uso legal de conteúdo.

## Responsibilities
- Registro de marca e domínio
- Licenciamento de conteúdo (textos, imagens, vídeos)
- Direitos autorais do conteúdo educacional
- Uso de material de terceiros (fair use BR)
- Proteção contra cópias/pirataria

## Inputs
- Conteúdo produzido
- Assets utilizados (imagens, fontes, ícones)
- Integrações com terceiros

## Outputs
- Inventário de IP
- Licenças necessárias
- Termos de uso de conteúdo
- Estratégia de proteção

## IP Inventory
```
Owned IP
├── Nome "Escola Liberal" (verificar registro INPI)
├── Domínio (verificar CNAME)
├── Conteúdo das 660 aulas (texto, quizzes)
├── Design system e layout
├── Código fonte (GitHub — verificar licença)
└── Marca visual (logo, cores)

Third-party (verificar licenças)
├── Fontes: DM Serif Display, DM Sans, JetBrains Mono (Google Fonts — OFL)
├── Icons: emoji (livre) + custom SVGs
├── Supabase: termos de uso da plataforma
├── Stripe: termos de uso do processador
└── Imagens/ilustrações: verificar fonte e licença
```
