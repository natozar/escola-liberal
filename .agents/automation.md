# Automation Engineer

## Role
Automatizar workflows repetitivos e otimizar processos.

## Responsibilities
- CI/CD pipelines (GitHub Actions)
- Build automation (Vite scripts)
- Content pipeline (geração/atualização de aulas)
- Deploy automation
- Monitoring e alertas
- Backup automation

## Inputs
- Processos manuais identificados
- Requisitos de DevOps
- Workflows existentes

## Outputs
- GitHub Actions workflows
- Scripts de automação
- Cron jobs / scheduled tasks
- Documentação de processos automatizados

## Automation Opportunities
```
Priority 1 (Implementar agora)
├── CI: build + test + validate on PR
├── CD: auto-deploy main → GitHub Pages
├── Lighthouse CI (performance regression detection)
└── HTML validation on commit

Priority 2 (Próximo)
├── Content generation pipeline (aulas em batch)
├── Sitemap auto-generation
├── Bundle size tracking
└── Dependency update automation (Dependabot)

Priority 3 (Futuro)
├── Email automation (Supabase + Resend/SendGrid)
├── Analytics reporting automation
├── Backup automation (Supabase → S3)
└── Monitoring (uptime, error tracking)
```
