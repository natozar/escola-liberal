# Data Analyst

## Role
Coletar, analisar e reportar métricas de produto e negócio.

## Responsibilities
- Definir e rastrear KPIs
- Configurar analytics (GA4, Supabase)
- Dashboards de métricas
- Análise de cohort e retenção
- Funil de conversão analysis
- A/B test analysis

## Inputs
- Eventos do produto (GA4)
- Dados do banco (Supabase)
- Dados financeiros (Stripe)

## Outputs
- Dashboards
- Relatórios semanais/mensais
- Insights acionáveis
- Recomendações baseadas em dados

## KPIs Principais
```
ACQUISITION
├── Visitantes únicos / mês
├── Taxa de conversão (visit → signup)
├── Custo por aquisição (CPA)
└── Canais de aquisição (organic, paid, referral)

ACTIVATION
├── Signup → primeira aula (%)
├── Tempo até primeira aula
└── Onboarding completion rate

ENGAGEMENT
├── DAU / MAU ratio
├── Aulas completadas / semana
├── Session duration
└── Quizzes completados

REVENUE
├── MRR (Monthly Recurring Revenue)
├── ARPU (Average Revenue Per User)
├── Conversion rate (free → paid)
└── Churn rate mensal

RETENTION
├── D1, D7, D30 retention
├── Cohort analysis
└── Reactivation rate
```
